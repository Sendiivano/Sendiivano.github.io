# sendiivano.github.io

Personal portfolio website built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean code.

## Themes

| Theme | URL | Description |
|-------|-----|-------------|
| Minimal | `/` | Clean, minimal light/dark design |
| Motorsport | `/motorsport` | Mercedes F1 inspired — black & teal |

## Structure
sendiivano.github.io/
├── index.html        # Minimal theme
├── photo.jpg         # Profile photo
└── motorsport/
└── index.html    # Motorsport theme\

## Built With

- Vanilla HTML, CSS, JavaScript
- [DM Sans & DM Serif Display](https://fonts.google.com) — Minimal theme
- [Barlow Condensed & Share Tech Mono](https://fonts.google.com) — Motorsport theme
- Hosted on [GitHub Pages](https://pages.github.com)

## Features

- Fully responsive
- Dark mode support (Minimal theme)
- Smooth scroll animations
- Custom cursor
- Theme switcher
- No build tools or dependencies
